## Custom Styles

The Custom Style feature provides the flexibility to apply custom styling to the vertical navigation menu when breakpoint is reached. With the `customStyles` prop, you can pass a set of CSS properties to customize the appearance of the navigation menu according to your design preferences. The `customStyles` prop accepts a CSSObject from the [Emotion](https://emotion.sh/docs/introduction) library.
