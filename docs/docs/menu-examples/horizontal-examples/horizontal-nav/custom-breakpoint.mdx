# Custom Breakpoint

:::warning
Please note that the `VerticalNavContent` used in this example is only for documentation purposes. If you want to use it in your own component, please import it from `src/components/layout/horizontal/VerticalNavContent.tsx` and use it accordingly.
:::

The `customBreakpoint` prop lets you choose your own size for when the navigation menu changes. Normally, this happens on a large screen (1200px), but you can set it to a different size if you want. This way, you don't have to use the default breakpoint values listed [here](/docs/menu-examples/horizontal-examples/horizontal-nav/breakpoint).

### Props

```ts noCopy
customBreakpoint?: string
```
