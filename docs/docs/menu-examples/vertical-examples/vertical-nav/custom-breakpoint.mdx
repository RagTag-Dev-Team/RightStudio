# Custom Breakpoint

The `customBreakpoint` prop lets you choose your own size for when the navigation menu changes. Normally, this happens on a large screen (1200px), but you can set it to a different size if you want. This way, you don't have to use the default breakpoint values listed [here](/docs/menu-examples/vertical-examples/vertical-nav/breakpoint).

### Props

```ts noCopy
customBreakpoint?: string
```
