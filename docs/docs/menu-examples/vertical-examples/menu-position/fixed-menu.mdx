# Fixed Menu

By default, the navigation sidebar is fixed and does not scroll with the content of the page. This means that as you scroll the page, the navigation sidebar remains in a fixed position. However, you can customize this behavior by using the `scrollWithContent` prop.

### Props

```ts noCopy
scrollWithContent?: boolean
```

```ts noCopy title='Default Value'
scrollWithContent={false}
```
