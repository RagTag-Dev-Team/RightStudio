<h1 className='!m-0'>Keen Slider</h1>

<code>keen-slider</code> is a third-party library. Please refer to its [official documentation](https://keen-slider.io/) for more details.

You may refer to [this](/docs/faqs/how-to-add-rtl-in-swiper) link on how to implement rtl support for <code>keen-slider</code>

:::warning
It is essential to ensure that when incorporating the <code>keen-slider</code> component into your project, it is correctly wrapped using the <code>AppKeenSlider</code> component. In scenarios where only a single instance of <code>keen-slider</code> is utilized, it should be individually enclosed within the <code>AppKeenSlider</code> component. Conversely, if multiple instances of <code>keen-slider</code> are used in one page, it becomes necessary to wrap the entire page with the <code>AppKeenSlider</code> component to ensure optimal styles.
:::
